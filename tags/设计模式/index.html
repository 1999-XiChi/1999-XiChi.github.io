  • 谈谈前端的设计模式

    2021-03-10 |
    • { 设计模式 }

    对于前端er来说，有些常见的设计模式需要掌握：

    • 创建型
      • 单例模式
      • 工厂模式
      • 原型模式
    • 结构型
      • 代理模式
    • 行为型
      • 观察者模式/订阅-发布模式

    其中构造型最为常见，它主要提供了一种在创建对象的同时隐藏创建逻辑的方式。

    PS：在考察Vue数据双向绑定时，可以提及代理模式和订阅-发布模式:

    vue 采用的是数据劫持结合发布和订阅者模式的方式，通过 Object.defineProperty 来劫持各个属性的 getter 和 setter，在数据变动时发布消息给订阅者，触发相应的监听回调
